# 页面制作

页面制作指的是将视觉设计师设计好的视觉稿文件实现成具体的HTML页面。页面制作一般包括`PhotoShop切图`、`页面实现`两个过程。

- `PhotoShop切图`指的是根据视觉稿文件将图片、按钮等资源保存成单独的文件，以供后面页面实现调用。
- `页面实现`指的是用HTML+CSS等技术将视觉稿实现成HTML静态页面。
